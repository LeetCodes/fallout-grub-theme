1. **Secure way:**
    - Download install script:  
    `wget -P /tmp https://github.com/leetcodes/allout-grub-theme-modified/raw/master/install.sh`
    - Review install script at `/tmp/install.sh`
    - Run it: `bash /tmp/install.sh`
